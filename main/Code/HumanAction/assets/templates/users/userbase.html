Human Action Recognition

The ultimate goal of this project is to develop a generalised framework by incorporating ‘path signatures’ and deep learning to interpret complex multi-dimensional streamed data of human actions. Efficient and effective representation of streamed body pose data which can be used generically for understanding and analysing the actions of the body. Efficient and robust ways to combine path signatures with cutting-edge deep learning models to produce state-of-the-art results in action classification.
